# Метод автоматического выявления противоправных действий по скелетной модели человека



### Ключевые компоненты

| Компонент | Назначение |
|---|---|
| Multi-Scale Temporal Conv | Параллельные ветки с ядрами 3/7/15 кадров |
| Cross-Person Attention | Моделирование взаимодействия между людьми |
| Temporal Transformer | Глобальный временной контекст через CLS-токен |
| Dual Head | Action Recognition (120 кл.) + Anomaly Score |

---

## Датасет

**NTU RGB+D 120** (2D скелеты)

- 113 945 видеозаписей, 120 классов действий
- 9 классов противоправных действий
- Протокол оценки: cross-subject (xsub)


---

## Установка

```bash
pip install -r requirements.txt
```

---

## Использование

### Обучение
```bash
python3 train.py
```

### Оценка на тестовой выборке
```bash
python3 eval.py
```

### Инференс одного семпла
```bash
python3 inference.py
```

### Полный пайплайн (видео → результат)
```bash
python3 pipeline.py путь/к/файлу
```

## Результаты

| Метрика | Значение |
|---|---|
| Top-1 Accuracy (120 классов) | 69.4% |
| Top-3 Accuracy | 83.5% |
| Anomaly F1 | 0.864 |
| ROC-AUC | 0.9907 |
| Recall (противоправные) | 0.96 |
